# 星幻AI

星幻AI 是一个面向本地运行场景的微信机器人项目，提供外部首页、控制台配置界面、角色卡管理、环境检测与本地服务控制等能力。

## 上游项目与开源说明

- 原项目： [KouriChat](https://github.com/KouriChat/KouriChat)
- 上游二开项目： [WeChatBot_WXAUTO_SE](https://github.com/iwyxdxl/WeChatBot_WXAUTO_SE)
- 当前项目：基于上述项目继续修改与扩展的再次二开版本

## 当前仓库包含内容

- 本地客户端程序
- 本地外部首页
- 本地控制台与配置编辑器
- 角色卡、提示词、表情包与本地运行相关资源

## 当前仓库不包含内容

- 服务器端管理员后台
- 私有部署配置
- 客户交付包

## 主要功能

1. 本地外部首页与控制台入口
2. Web 控制台配置编辑
3. Chat 模型配置与辅助模型配置
4. 角色卡管理与分组
5. 图片 / 表情包识别
6. 联网搜索
7. 定时任务、提醒与自动消息
8. 记忆管理
9. 运行日志查看
10. 本地服务状态检测与控制

## 使用前准备

1. 安装 Python 与 pip
2. 登录 Windows 微信并保持可用
3. 按需准备模型 API Key
4. 根据自己的使用场景修改 [config.py](./config.py)

## 快速开始

1. 运行 `Run.bat`
2. 等待依赖检测与安装完成
3. 在浏览器中打开本地控制台
4. 按需填写模型配置、用户列表与角色卡
5. 完成配置后启动 Bot
